# NES-GO

Another NES Emulator implemented in Golang

# Dependencies

MacOS
```
brew install glfw
```

Linux/Unix Based Systems or Windows

Refer to https://www.glfw.org/


# Usage

```
go run cmd/nesgo/main.go <path-to-rom>
```
